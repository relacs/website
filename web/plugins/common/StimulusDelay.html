StimulusDelay   [Common Plugins]

Measures delays between actual and reported onset of a stimulus.

Author:
Jan Benda
Version:
1.2 (Feb 8, 2008)
  1. check interrupt() after each sleep

1.1 (Jan 10, 2008)

  1. removed stop() function
  2. moved class variables to the main() functions

1.0 (Dec 20, 2007)

Options
  • intrace (text): Name of the input trace.
  • outtrace (text): Name of the output trace.
  • samplerate (number, kHz): Sampling rate used for output of stimulus.
  • duration (number, ms): Duration of stimulus.
  • repeats (integer): Number of repetitions.
Files
  • No output files.
Plots
  • The read in stimulus aligned to the reported stimulus onset.
Requirements
  • The output must be connected to the input.
