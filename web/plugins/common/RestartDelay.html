RestartDelay   [Common Plugins]

Measures delays due to restart of the data acquisition.

Author:
Jan Benda
Version:
1.3 (Feb 8, 2008)
  1. removed subsample option
  2. check interrupt() after each sleep

1.2 (Jan 10, 2008)

  1. removed stop() function
  2. moved class variables to the main() functions

1.1 (Dec 20, 2007)

Options
  • duration (number, ms): Length of analysis window to the left and right of the restart event.
  • repeats (integer): Number of repetitions.
  • intrace (text): Name of the input trace.
  • samplerate (number, kHz): Sampling rate used for output of the zero stimulus.
Files
  • No output files.
Plots
  • The input sine wave (green) with the fits (red).
Requirements
  • A sine wave generator connected to the selected input channel.
