CalibSpeakers   [Auditory Plugins]

Calibrating attenuation for loudspeakers.

Author:
Jan Benda
Version:
1.1 (Jan 10, 2008)
  1. removed stop() function
  2. moved class variables to the main() functions

1.0 (Sep 1, 2004)

Bug:
too many values pushed into Gains and Offsets?
Calibrates attenuators such that a full amplitude sine wave is put out at an sound pressure level as requested by setIntensity(). The sound pressure level is base on the root mean square of the sound waveform!

Options
  • reset (boolean): Reset calibration, i.e. do not use old calibration results.
  • am (boolean): Use amplitude modulation signal instead of sine wave.
  • frequency (number, Hz): If there is no fish EOD, then use sine wave with this frequency as a calibration stimulus.
  • beatfreq (number, Hz): If there is a fish EOD, then use calibration stimulus that results in this beat frequency.
  • duration (number, ms): Duration of calibration stimulus.
  • pause (number, ms): Pause between successive stimuli.
  • maxcontrast (number, %): Maximum contrast (beat amplitude / EOD amplitude) to be used.
  • maxint (integer): Maximum number of test intensities.
  • minintensity (number, %): Minimum stimulus intensity relative to EOD amplitude.
  • repeats (integer): Maximum repetitions of the calibration protocol.
Files
  • calibrate.dat : the calibration data (measured versus requested stimulus intensity).
Plots
The plot shows the measured versus the requested stimulus intensity (red circles). The yellow line is a fit of a straight line to the data. This line should for a successful calibration coincide with the blue 1:1 line.
Requirements
  • Transdermal EOD recording (EODTrace2) and events (EODEvents2).
