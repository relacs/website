The following options are supported:
  • intensity=80dB: Peak intensity of the search stimulus (number)
  • duration=50ms: Duration of the search stimulus (number)
  • pause=500ms: Duration of the pause between two succeeding stimuli (number)
  • prepause=100ms: Part of the pause before stimulus (number)
  • frequency=5kHz: Frequency of the stimulus (number)
  • waveform=sine|noise: Sine-wave or white-noise stimulus (text)
  • ramp=2ms: Ramp for stimulus onset and offset (number)
  • side=left|right|best: Loudspeaker used for output of the stimulus (text)
  • repeats=0: Number of repetitions of the search stimulus. 0 = infinitely (integer)
  • adjust=true: Adjust input gains? (boolean)
  • setbestside=never|no session|alwayse: Set the "best side" variable of the session to the current used speaker (text)
  • keep=true: Keep changes that have been entered through the RePro's GUI (boolean)

Keyboard shortcuts:

  • Left: Left speaker
  • Right: Right speaker
  • M, Pause: Toggle mute
  • Up: Increase intensity by 1dB
  • Down: Decrease intensity by 1dB
  • Shift+Up: Increase intensity by 5dB
  • Shift+Down: Decrease intensity by 5dB
  • Alt+Up: Increase frequency by 1kHz
  • Alt+Down: Decrease frequency by 1kHz
  • Alt+Shift+Up: Increase frequency by 5kHz
  • Alt+Shift+Down: Decrease frequency by 5kHz
