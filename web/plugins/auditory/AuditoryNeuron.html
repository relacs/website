AuditoryNeuron   [Auditory Plugins]

Spiking neuron model stimulated through an auditory transduction chain.

Author:
Alexander Wolf

Jan Benda

Version:
1.1 (Jan 15, 2006)
Auditoryneuron simulates the auditory transduction chain of Locusta migratoria. It is designed to work with RELACS in the dry mode (relacs -3) so new repros can be tested for different settings. The plugin is based on a biophysical model that describes the transformation of the sound stimulus in a chain of three distinct steps: 1. the tympanum; 2. mechano-electrical transduction; 3. spike generator.

For each of the three steps, the plugin provides several options for the user to adapt to his/her needs and the hardware restrictions. Additionally, the user can chose between between several integration algorithms. The following options can be selected (in order of increasing complexity):

1. Tympanum

  • None: No modification of the stimulus
  • Scaling: Adjusts the stimulus strength according to tuning curve
  • Oscillator: Model as damped harmonic oscillator
2. Mechano-Electrical Transduction:
  • None: No modification of the tympanum output
  • Linear: Absolut value of the tympanum output
  • Linear saturated: Absolut value up to saturation point
  • Box: Two Heavyside functions
  • Square: Square of the tympanum output
  • Square saturated: Square value up to saturation point
  • Linear Boltzman: Superposition of two Boltzman functions with linear dependency in exponent
  • Square Boltzman: One Boltzman function with quadratic dependency in exponent
3. Spike Generators:
  • Stimulus: No modification of previous function
  • FitzhughNagumo: Simplified abstract neuron model with 2 differential equations.
  • MorrisLecar: Simplified neuron model based on ion channels with 2 differential equations.
  • WangAdapt: Neuron Model with adaptation currents with 5 differential equations.
4. Integrators:
  • Euler
  • Midpoint
  • Runge-Kutta 4th order
The following options are supported (brief description of each option with default values and data type):
  • tymp=Scaling: Select model for tympanum (1.) (integer)
  • freq=5.0kHz: Eigenfrequency of the tympanal membrane (number)
  • tdec=0.154ms: Damping time constant of tympanal membrane (number)
  • nl=Square saturated: Select model for mechano-electrical transduction (2.) (integer)
  • imax=60.0: Saturation = maximum current (number)
  • imin=0.0: Minimum = Zeron point current(number)
  • cut=0.02mPa: Amplitude where imax is reached (number)
  • slope=3000mPa^-1: Slope of linear Boltzman (number)
  • matchslope=1: Match slope of Boltzman to that of Square (boolean)
  • slope2=600000: Slope of square Boltzman (number)
  • matchslope2=1: Match slope of square Boltzman to that of Square (boolean)
  • spikemodel=: Select spikemodel (3.) (integer)
  • noise=0.0: Standard deviation of current noise (number)
  • deltat: Integration time constant (step size) (number)
  • integrator=Euler: Method of integration (4.) (integer)
  • Many more options for spike generators.
